keyboard-layout
===============

my keyboard layout configuration

key features
------------

* portable AutoHotkey
* Hotkey remap <b>AND</b> Hotstring-like functionality with partial support for backspace

AnyKey
------

is the function that enables complex partial 'Hotstrings'

    transs := [["ue", "ü"], ["alue", "alue"]]

"Nuesse" -> "Nüsse"<br/>
"value" -> "value"
